# Homebrew Tap

Custom Homebrew formulae.

## Usage

```bash
brew tap rvben/tap
```

## Available Formulae

### ipcam

A CLI for managing IP cameras (Tapo, Reolink) via RTSP and vendor APIs.

```bash
brew install rvben/tap/ipcam
```

### shelly-cli

A CLI for controlling Shelly smart home devices.

```bash
brew install rvben/tap/shelly-cli
```

### confluence-cli

A Markdown-sync-first Confluence CLI for Cloud and Data Center.

```bash
brew install rvben/tap/confluence-cli
```
